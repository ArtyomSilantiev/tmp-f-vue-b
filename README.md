# Description

Project Template (Frontend)

Frontend technologies:
+ vue2 
+ composition api 
+ typescript
+ bootstrap
+ Custom Libs

# Frontend first deploy
```sh
cd <tmp-f-vue-root>
cd npm i && npm run build
```

# Frontend watch
```sh
cd <tmp-f-vue-root>
npm run serve
```

# Frontend build
```sh
cd <tmp-f-vue-root>
npm run build
```
